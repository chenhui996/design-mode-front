# 模版方法模式

- 封装内部多个 API，实现对外暴露少量 API。

## 示例

- 将多个内部的顺序处理 API，做封装。
  - 封装后，仅暴露一个或少量 API。
    - 只需调用一个或少量 API，即可依次执行多步顺序操作。
