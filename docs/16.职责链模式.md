# 职责链模式

- 一步操作可能分为 **多个职责角色** 来完成。
- 将这些 **角色拆分**，用一个 **链** 串联。
- 将 **发起者** 和 **各个处理者** 进行隔离。

## 示例

- 请假条，多人审批。
