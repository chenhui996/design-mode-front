# 代理模式

- 使用者无权访问目标对象。
- 中间加代理，**通过代理做授权和控制**。

## 示例

- 科学上网、魔法丝袜。
- 在家办公，访问公司内网。（需要连到公司给的代理地址）
- 明星的经纪人。
