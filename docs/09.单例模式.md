# 单例模式

- 系统中被唯一使用。
- 一个类只有一个实例。

## 示例

- 登录框
- 购物车

## 说明

- 单例模式需要用到 java 的特性（private）。
- ES6 中没有（typescript 除外）。
- 可用 java 代码来演示 UML 图的内容。

## UML 类图

- 略

## ES6 语法最简易模型

- 代码目录：`./src/09.单例模式-demo.js`

## 经典使用场景

- jQuery 只有一个 $
- 模拟登录框
- 其他：
  - 购物车（和登录框类似）
  - vuex 和 redux 中的 store
