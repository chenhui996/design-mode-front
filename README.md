# 前端设计模式 - 实训

- 实现一个简易的购物车组件。

## 用到的设计模式

- 工厂模式
- 单例模式
- 装饰器模式
- 观察者模式
- 状态模式
- 模版方法模式
- 代理模式

> 说明：一个示例很难所有设计模式都用到。不可仅为写设计模式，而强行用设计模式。

## 总结用到的设计模式 - 功能

- 工厂模式： $('xxx')，创建商品
- 单例模式： 购物车
- 装饰器模式： 打点统计
- 观察者模式： 网页事件，Promise
- 状态模式： 添加到购物车 & 从购物车删除
- 模版方法模式： 渲染有统一的方法，内部包含了各模块渲染
- 代理模式： 打折商品信息处理
